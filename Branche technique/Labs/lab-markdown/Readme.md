
# Lab-Markdown


## Travail a faire

exercice : Créer une page markdown 

* créez un fichier Readme.md
* Utilisez la syntax markdown pour créer une page markdown  avec les éléments suivant :
  * Headings
  * text-styling
  * List
  * Link
  * Image
  * Code block
  * Tables
  * Check Box List
  * Horizontal Line
  * text UP-DOWN



### Markdown Styling Commands:

1. **Headers**
```css
# h1
## h2
### h3
```
1. **text-styling**
```css
**BoldText**
*ItalicText*
```
1. **List**
````css
* list 1
* list 2
    * SubList 2.1
1. Ordered List 
2. Ordered List 
````
1. **Link**
```css
[text](https://www.example.com)
```
1. **Images**
```css
![Alt](image-url)
```
1. **Code block**
```js
Use three backticks (```) for code block. Optionally, specify the programming language after the opening three backticks for syntax highlighting.
```
1. **Tables**
```js
| Header 1 | Header 2 |
|----------|----------|
| Row 1    | Row 1    |
| Row 2    | Row 2    |
```
1. **Check Box List**
```js
- [x] Task 1
- [x] Task 2
- [ ] Task 3
```

1. **Horizontal Line** 
```js
---
OR
___
```
1.  **text UP-DOWN & strike-text**
```css
^UP^
~DOWN~
~~strike~~
```
